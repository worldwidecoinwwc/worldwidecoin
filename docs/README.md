# WorldWideCoin GitHub Pages

This directory contains the GitHub Pages website for WorldWideCoin cryptocurrency.

## Website Features

- **Professional Design**: Modern, responsive web interface
- **Real-time Statistics**: Live blockchain data display
- **Mining Information**: Complete mining specifications
- **Download Links**: Easy access to software releases
- **Whitepaper Access**: Direct link to professional documentation
- **Network Status**: Current blockchain and mining status

## File Structure

```
docs/
  index.html          # Main website page
  README.md           # This file
  assets/             # Static assets (CSS, JS, images)
```

## Deployment

The website is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

## Customization

To customize the website:

1. Edit `index.html` for content changes
2. Add assets to the `assets/` directory for additional resources
3. Update statistics and information as needed
