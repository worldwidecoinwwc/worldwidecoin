# WorldWideCoin (WWC)

A decentralized cryptocurrency with fair distribution, predictable monetary policy, and long-term sustainability.

## Features

- **Fair Launch**: No premine, no ICO, mining-only issuance
- **Continuous Decay**: Smooth 2.5% annual reward reduction (no abrupt halvings)
- **CPU-Friendly**: SHA-256 mining accessible to everyone
- **Treasury System**: 5% automatic allocation for development
- **Fee Burn**: 20% transaction fee burn for deflation
- **Whitepaper Compliant**: Following professional specifications
