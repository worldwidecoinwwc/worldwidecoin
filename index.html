Get Started with Mobile App

Prerequisites: Node.js 16.x+, React Native CLI, Android Studio (Android) or Xcode (iOS)

Installation:

# Clone the repository
git clone https://github.com/worldwidecoinwwc/worldwidecoin.git

# Navigate to mobile directory
cd worldwidecoin/mobile

# Install dependencies
npm install

# Run on Android
npm run android

# Run on iOS (macOS only)
npm run ios

Download: React Native project is available in the mobile/ directory of the repository.

API Quick Start

Base URL: http://localhost:8080/api/v1

Example Request:

curl -X GET "http://localhost:8080/api/v1/blockchain/info"
-H "Content-Type: application/json"

Response:

{
  "success": true,
  "data": {
    "height": 12345,
    "difficulty": 1.23,
    "hash_rate": 1234567,
    "total_supply": 21000000,
    "block_reward": 50
  }
}
